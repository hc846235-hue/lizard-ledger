# CloudBase 绑定自定义域名指南

## 前提准备

在开始之前，你需要：

1. **已备案的域名**（如果是国内域名）
2. **域名管理权限**（能够修改 DNS 记录）
3. **CloudBase 环境访问权限**

---

## 方案一：绑定自定义域名（推荐）

### 步骤 1：登录 CloudBase 控制台

1. 访问：https://console.cloud.tencent.com/tcb
2. 选择你的环境：`mm223-7gozbhmt7b381a50`
3. 进入左侧菜单「静态网站托管」

### 步骤 2：添加域名

1. 点击「添加域名」按钮
2. 输入你的自定义域名，例如：
   - `ledger.yourdomain.com`（子域名）
   - 或 `www.yourdomain.com`（主域名）

### 步骤 3：配置 DNS 解析

CloudBase 会给你提供 DNS 记录信息，通常如下：

```
记录类型：CNAME
主机记录：@ 或 ledger
记录值：mm223-7gozbhmt7b381a50-1302094821.tcloudbaseapp.com
```

**配置方法：**

1. 登录你的域名服务商（如阿里云、腾讯云、GoDaddy 等）
2. 找到域名 DNS 管理页面
3. 添加以下记录：

| 记录类型 | 主机记录 | 记录值 | TTL |
|---------|---------|--------|-----|
| CNAME   | @       | mm223-7gozbhmt7b381a50-1302094821.tcloudbaseapp.com | 600 |
| CNAME   | www     | mm223-7gozbhmt7b381a50-1302094821.tcloudbaseapp.com | 600 |

**说明：**
- 如果是子域名（如 `ledger.yourdomain.com`），主机记录填写 `ledger`
- 如果是主域名（如 `yourdomain.com`），主机记录填写 `@`
- 同时添加 `www` 记录可以支持 `www.yourdomain.com` 访问

### 步骤 4：等待 DNS 生效

DNS 解析通常需要 10 分钟到 24 小时生效：
- **国内域名**：通常 10-30 分钟
- **国际域名**：可能需要 1-24 小时

### 步骤 5：验证域名绑定

1. 在 CloudBase 控制台查看域名状态
2. 状态变为「已启用」表示绑定成功
3. 访问你的自定义域名测试

### 步骤 6：更新 Vite 配置（可选）

如果你希望自定义域名直接访问根路径，可以修改 `vite.config.ts`：

```typescript
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'

export default defineConfig({
  plugins: [react()],
  // 如果使用自定义域名作为根路径，可以改为 '/'
  base: '/',
})
```

然后重新构建并上传：
```bash
npm run build
```

## 方案二：升级为正式环境（付费）

### 步骤 1：了解套餐对比

| 套餐类型 | 流量限制 | 静态托管 | 价格 |
|---------|---------|---------|------|
| 免费版 | 5GB/月 | 测试域名 | ¥0 |
| 基础版 | 100GB/月 | 支持自定义域名 | ¥29/月 |
| 专业版 | 500GB/月 | 支持自定义域名+CDN | ¥199/月 |

### 步骤 2：升级套餐

1. 在 CloudBase 控制台，进入「环境概览」
2. 点击「升级」或「购买套餐」
3. 选择适合的套餐（建议基础版起步）
4. 完成支付

### 步骤 3：升级后配置

升级后，免费测试域名会被替换为正式域名，但仍建议绑定自定义域名。

## 域名备案说明（国内域名）

### 必须备案的情况

如果满足以下所有条件，必须完成 ICP 备案：
1. 服务器在中国大陆（CloudBase 是腾讯云国内服务）
2. 使用国内域名（如 `.cn`、`.com.cn` 等）
3. 提供公开访问的 Web 服务

### 备案流程

1. **准备材料**：
   - 营业执照
   - 法人身份证
   - 网站负责人身份证
   - 服务器提供商账号（腾讯云）

2. **提交备案**：
   - 登录腾讯云备案系统：https://console.cloud.tencent.com/beian
   - 按照指引填写信息
   - 上传证件照片

3. **等待审核**：
   - 腾讯云初审：1-2 个工作日
   - 管局审核：7-20 个工作日

4. **备案成功后**：
   - 获得备案号（如：粤ICP备12345678号）
   - 在网站底部添加备案号

### 可以不备案的情况

- 使用国际域名（`.com`、`.net`、`.org`）+ 境外服务器（CloudBase 有香港节点）
- 仅用于个人测试，不公开访问

## HTTPS 证书配置

绑定域名后，CloudBase 会自动免费提供 HTTPS 证书：

1. CloudBase 会自动申请 Let's Encrypt 证书
2. 证书自动续期，无需手动管理
3. 支持 HTTP 自动跳转 HTTPS

## 推荐方案对比

| 方案 | 优点 | 缺点 | 适用场景 |
|-----|------|------|---------|
| **GitHub Pages** | 免费、无流量限制 | 国内访问慢、功能有限 | 个人项目、开源项目 |
| **CloudBase + 自定义域名** | 国内访问快、功能全 | 需要备案、可能产生费用 | 商业项目、正式上线 |
| **CloudBase 升级付费版** | 无流量限制、支持 CDN | 需要付费 | 高流量项目 |
